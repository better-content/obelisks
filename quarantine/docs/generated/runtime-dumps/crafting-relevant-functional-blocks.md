# Crafting-Relevant Functional Blocks

Reduced from the broad functional audit to blocks that either act as a player-facing crafting/processing surface, are block ingredients directly used to craft one of those surfaces, or are block ingredients one recipe step upstream in a component used by one of those surfaces.

This intentionally excludes storage-only, redstone-only, transport-only, portal, signal, lamp, seat, decorative table, decorative plant, and generic utility blocks unless the recipe graph puts them directly in the construction path for a crafting/process surface. The source registry/tag dumps are currently empty, so classification remains keyword/blockstate based.

## Counts
- total reduced blocks: `224`
- crafting/process surfaces: `171`
- direct block ingredients for surfaces: `42`
- one-step block ingredients for surface components: `47`

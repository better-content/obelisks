# Exhaustive Realistic Hands Tooling

These inputs generated and exempted the retired block-by-block Realistic Hands policy. They are
preserved for reference only and are not part of the supported `tools/bc` surface.

`retired-log-tag-overrides/` preserves the pack-wide block and item `minecraft:logs` and
`logs_that_burn` overrides that fed the old matrix. Their hard references to absent content
invalidated the standard Minecraft log tags. The active pack files now contribute only the eight
stable vanilla wood-family tags; vanilla and installed mods own the remaining membership.

The active audit and validator cover only the no-tree-punching log gate.
